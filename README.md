# The Walls Minigame
## About
This is a bukkit plugin made for minecraft 1.8, used to replicate a classic Minecraft map called "The Walls" made by Hypixel.
Some Features:
* multiple arena support
* scoreboards
* 4 teams
* configurable spawn points
